# 経費申請アプリ

ブラウザだけで動作する、サーバー不要の経費申請アプリです。`expense-app/index.html`
を開くだけで使えます（ビルド不要）。

## 主な機能

### 1. レシート画像の自動読み取り（画像解析）

- レシート・領収書の写真をアップロード（クリック選択またはドラッグ＆ドロップ）
- [Tesseract.js](https://tesseract.projectnaptha.com/) による **端末内OCR** で文字を認識
  （画像やテキストは外部サーバーへ送信されません）
- 認識結果から **金額（合計）・日付・支払先（店名）** を推定し、申請フォームへ自動入力
- 読み取ったテキスト全文も確認可能。抽出結果は手修正できます

### 2. 経費の申請

- 日付・科目・支払先・金額・摘要を入力して申請
- 科目：交通費／交際費／会議費／消耗品費／通信費／宿泊費／その他
- 申請内容とレシート画像のサムネイルはブラウザの localStorage に保存

### 3. ダッシュボード（個人確認用・管理者確認用で分離）

- **個人ダッシュボード**：ログインユーザー（氏名）自身の申請一覧と、
  申請件数・申請中／承認済み金額の集計。申請中のものは取消可能
- **管理者ダッシュボード**（画面右上「管理者モード」で表示）：全申請の一覧、
  承認待ち件数・金額、科目別の承認済み金額グラフ、申請者・店名での検索。
  各申請の **承認 / 却下（理由入力）/ 差戻** を操作可能

## 使い方

1. `expense-app/index.html` をブラウザで開く（公開URLでも可）
2. 画面右上の「ログインユーザー」に氏名を入力
3. 「経費を申請」タブでレシート画像をアップロード → 自動入力された内容を確認して申請
4. 「個人ダッシュボード」で自分の申請状況を確認
5. 承認担当者は「管理者モード」をONにし、「管理者ダッシュボード」で承認／却下

## 実装メモ

- フレームワーク不要のバニラ JS（`index.html` / `style.css` / `script.js`）
- データは各ブラウザの localStorage に保存されるため、端末・ブラウザをまたいだ
  共有はされません（社内共有が必要な場合はバックエンド連携が別途必要です）
- OCR ライブラリは CDN から読み込みます。初回解析時に言語データのダウンロードが
  走るため、ネットワーク接続が必要です
