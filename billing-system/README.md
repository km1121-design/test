## GAS のセットアップ

1. <https://script.google.com> で新規プロジェクトを作成し、`apps-script/*.gs` を貼り付け
2. スクリプトプロパティ（任意）:
   - `SOURCE_SPREADSHEET_ID` … 実績・経費シートID（既定は実データID）
   - `OUTPUT_SPREADSHEET_ID` … 中間データの出力先（未設定なら新規作成）
3. `runStep1()` を実行 → タブ一覧・コースインベントリを出力

> ★ ソース（実績・経費シート／シフトシート）は**読み取り専用**。書き込みは別ファイルへ。
